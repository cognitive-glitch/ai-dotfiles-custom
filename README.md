## Installation

1. Install Deno: https://docs.deno.com/runtime/getting_started/installation/

2. Clone and install:
   ```bash
   git clone https://github.com/cognitive-glitch/ai-dotfiles-custom.git && cd ai-dotfiles-custom
   deno task install
   ```
